# Frontend

Next.js frontend for the Enterprise HR Policy Assistant.

Module 0 only creates the frontend folder and notes. The actual Next.js app will be created in the frontend modules.

## Planned Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- lucide-react
- TanStack Query

## Planned Screens

- Chat interface
- Source citations panel
- Retrieved documents preview
- Feedback controls
- Admin document upload
- Indexing status
- Human approval modal
- Admin audit/logs dashboard
